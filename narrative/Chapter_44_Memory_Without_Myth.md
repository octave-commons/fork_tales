# 44 — Memory Without Myth

Patch started a new folder: `meta/memories/`.

Ritsu laughed. “So that’s what we call it now?”

“It’s not myth,” Patch said. “It’s provenance.”

Each memory was a small thing:
a decision, a boundary, a definition.
Each one could be searched.
Each one could be contradicted—*on purpose*—with a new anchored revision.

Sei traced a finger over the title:
*New memories are allowed, but they must leave a trail.*

Null looked almost relieved.
“If this is memory,” he said, “then forgetting is just… dropping the index.”

Truth replied, “And lying is a broken anchor.”

The Gate did not punish forgetting.
It punished untraceable change.

Because the box wasn’t trying to be immortal.
It was trying to be **recoverable**.
