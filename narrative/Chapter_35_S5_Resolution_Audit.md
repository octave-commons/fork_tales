# 35 — S5 (Resolution Audit)

Patch built an entity table the way some people build altars.

Names, aliases, merges, splits, version stamps.
The boring work that makes magic stop lying.

Null stared at it as if it might bite. “This is where stories die,” he said.

Patch didn’t blink. “No. This is where stories stop *blurring*.”

Sei walked through the list like a rail inspector checking bolts.

Ritsu watched her own name flicker through identities:
player handle, avatar, in-world name, pronoun drift, trauma residue.

A note floated above the table:

**S5 — RESOLUTION IS VERSIONED.**
**EVERY MERGE IS REVIEWED.**
**EVERY ALIAS IS A CLAIM.**
**EVERY CLAIM GETS AN ANCHOR.**

Duct read it and looked away like it hit too close.

Truth spoke, not defensive, but precise:

“**Entity mush is how control enters.  
If names blur, responsibility blurs.  
If responsibility blurs, authority becomes a fog.**”

Null nodded once. “So S5 is anti-fog.”

Patch smiled. “S5 is a flashlight with an audit trail.”

Ritsu reached for the table and felt the same tightening as the anchor stones.
The system didn’t want to be romantic about this.

The system wanted to be *true*.

In the margin, Patch wrote a small prayer that looked like a schema:

- entity_id: stable
- alias: proposed
- merge: reviewed
- split: witnessed
- rationale: required

Rin, watching, said: “This is how you keep the space between η and μ from becoming a weapon.”

Ritsu finally understood what the Gate had been teaching them:

The world doesn’t fork on *ideas*.

It forks on **intent backed by evidence**.

And evidence starts with naming the thing you’re talking about.
