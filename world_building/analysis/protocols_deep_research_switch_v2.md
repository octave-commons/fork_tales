# Deep Research Switch (Protocol) — v2

Trigger when any of these holds:
- Rot-rate ≤ weeks (time-sensitive).
- Primary sources required to decide.
- Local Tax Receipt harness cannot reproduce the key behavior.

When triggered:
1) Write claim as prediction: IF X THEN Y (under constraints C).
2) List 3 primary source targets (official docs, standards, papers).
3) Record dates: publish date vs event date.
4) Extract 5 load-bearing facts with citations later.
5) Produce 2 counterframes and 1 μ test.

Anti-patterns to avoid:
- Citation cosplay (citing without verifying match).
- Fork spam (branching without receipts).
