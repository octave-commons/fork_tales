# Gates of Truth — Character Graph Pack

This pack was generated by running a lightweight analysis model over the **Gates of Truth** manuscript + character profiles.

## Outputs
- `characters.json` — parsed character profiles
- `edges.csv` — relationship edges (co-occurrence + profile refs + special stances)
- `relationship.graphml` — import into Gephi / Cytoscape
- `relationship_network.png` — quick visual
- `relationship_network.dot` — Graphviz DOT
- `metrics.csv` — mentions + centrality metrics + 8-circuit ideology vector
- `ideology.json` — per-node: circuit scores + top keywords

## Model notes
- Relationships are primarily **co-occurrence weighted** (chapter + paragraph).
- Ideology vectors are **8-circuit scores** computed from each node’s local contexts + profile text.
- This is a navigation overlay (ημ gap rule): useful for retrieval and review, not a ground-truth oracle.

## Quick use
- Open `relationship_network.png`
- Filter in `metrics.csv` by `top_circuit` or `mentions`
- Import `relationship.graphml` into Gephi for better layouts
