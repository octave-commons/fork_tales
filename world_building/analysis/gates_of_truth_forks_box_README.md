# Gates of Truth — Forkgate (Local Box)

Forks are gated and ranked to prevent “forks everywhere”.

Outputs:
- `analysis/gates_of_truth_forks_box/fork_candidates.jsonl`
- `analysis/gates_of_truth_forks_box/anchors.jsonl`
- `analysis/gates_of_truth_forks_box/timeline_forks.md`

Every fork is anchored to an underlying intent event snippet hash.
