# 1️⃣ Micro Pitch Modulation (The Biggest Win)

Human speech has tiny pitch fluctuations. TTS often doesn’t.

Instead of a constant pitch shift, add *subtle modulation*.

### Option A – Rubberband (cleanest)

```bash
ffmpeg -i input.wav -af "rubberband=pitch=1.02" temp.wav
```

Then:

```bash
ffmpeg -i temp.wav -af "rubberband=pitch=0.99" output.wav
```

Why two passes?
It introduces micro-instability without sounding artificial.

---

### Option B – Vibrato (very subtle)

```bash
ffmpeg -i input.wav -af "vibrato=f=4.5:d=0.2" output.wav
```

Keep depth LOW (0.1–0.2).
Higher makes it sound like a singer.

---

# 2️⃣ Add Dynamic Movement

Bored = flat amplitude envelope.

Use upward compression + mild expansion:

```bash
ffmpeg -i input.wav -af "acompressor=threshold=-24dB:ratio=1.6:attack=3:release=80" temp.wav
```

Then:

```bash
ffmpeg -i temp.wav -af "alimiter=limit=-1dB" output.wav
```

This increases presence without crushing it.

If you want more energy:

```bash
ratio=2.0
```

Don’t exceed 2.2 or it becomes radio-voice.

---

# 3️⃣ Brightness / Presence Boost

Melo voices are slightly dull around 2–5kHz.

Add:

```bash
ffmpeg -i input.wav -af "equalizer=f=3500:t=q:w=1.2:g=3" output.wav
```

Optional add:

```bash
equalizer=f=6000:t=q:w=1:g=2
```

That gives articulation and perceived alertness.

---

# 4️⃣ Add Transient Excitement (Secret Weapon)

A tiny harmonic exciter effect:

```bash
ffmpeg -i input.wav -af "aexciter=amount=2:drive=3" output.wav
```

This adds harmonic richness.

Too much makes it harsh — keep amount low.

---

# 5️⃣ Stereo Widen Slightly (Psychological Trick)

Even mono voices sound more alive when widened subtly:

```bash
ffmpeg -i input.wav -af "stereotools=mlev=0.02" output.wav
```

Very subtle width creates spatial energy.
