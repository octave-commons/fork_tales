# 🎯 Strategy Overview

We will align:

1. Pitch center + variance
2. Spectral tilt (brightness/body)
3. Dynamic envelope
4. Transient sharpness
5. Room/mic illusion
6. Emotional intensity curve

All subtle. Stacked.

---

# 1️⃣ Normalize Pitch Center AND Range

Not just average pitch — range too.

If JP is higher *and* more expressive:

### Step A: Align mean F0

Move both toward a shared midpoint (not identical).

If you can’t access F0 directly:

* JP: lower 3–5%
* EN: raise 2–3%

```bash
rubberband=pitch=0.97   # JP
rubberband=pitch=1.02   # EN
```

---

### Step B: Match pitch variance

If JP feels more animated:

* Increase EN pitch variance slightly.
* Decrease JP pitch variance slightly.

If you have F0:

```python
f0 = mean + (f0 - mean) * 1.1   # expand EN
f0 = mean + (f0 - mean) * 0.95  # tame JP
```

This is crucial. Mismatch in pitch variance screams “different person.”

---

# 2️⃣ Force Shared Spectral Identity (Very Important)

Forget boosting each separately.

Design one EQ curve and apply it to BOTH.

Example “Narrator EQ”:

```bash
equalizer=f=160:t=q:w=1.0:g=1.5,
equalizer=f=3200:t=q:w=1.2:g=2,
equalizer=f=7000:t=q:w=1:g=1
```

This becomes your **house tone**.

Even if the base voices differ, the shared curve glues them.

---

# 3️⃣ Match Dynamic Character

Different models compress differently.

Apply the same compressor to both:

```bash
acompressor=threshold=-22dB:ratio=1.6:attack=8:release=120
```

This forces similar micro-dynamics.

---

# 4️⃣ Unify “Mic Character”

This is often what people describe as “they feel like different rooms.”

Add a tiny shared room IR.

Wet: 4–6%.

Even better:

Add 10–15ms pre-delay before reverb.
This makes it feel intentional, like one narrator.

---

# 5️⃣ Match Energy Curve

If JP sounds emotionally more alive and EN sounds restrained:

Do NOT boost EN globally.

Instead apply envelope exaggeration equally:

```python
scale = 1.0 + (envelope - envelope.mean()) * 0.12
audio *= scale
```

Apply same algorithm to both.

Now they share emotional movement shape.

---

# 6️⃣ Smooth the Language Switch Itself

Hard switching is perceptually violent.

When switching languages:

* Fade out last 50ms of EN
* Add 80–120ms silence
* Fade in JP over 40–60ms

This alone reduces perceived mismatch by 30–40%.

---

# 7️⃣ Final Glue: Shared Harmonic Signature

Add very subtle harmonic enhancement to both:

```bash
aexciter=amount=1.2:drive=2
```

Not to make it bright — to make harmonic structure similar.
